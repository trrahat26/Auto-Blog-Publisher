How to use (Blogger):
1) Header logo: upload dbp-logo-horizontal.png in Layout -> Header.
2) Favicon: upload favicon.ico or dbp-icon-32.png in Settings -> Favicon.
3) Social profile image: use dbp-icon-512.png.
4) Brand page/hero blocks: use dbp-logo-stacked.png.

Color notes:
- Primary green: #2F6F4F
- Beige accent (readable): #A88760
- Neutral text: #1B1B1B
